Привет! Вы находитесь на удалённом репозитории игры LABaria. Здесь вы можете видеть папки с текстурами, анимацией, папку со шрифтом и папку с сохранениями, а также модули игры. 
Самый важный модуль - main.py. В этом модуле запускается игра. К нему вернёмся чуть позже.
Модуль block.py содержит класс Block. Из блоков состоит игровой мир.
Модуль world.py содержит класс World. При создании объекта класса World ландшафт либо генерируется, либо загружается из файла. Кроме генерации в модуле содержится проверка столкновения мира с игроком и функции, позволяющие ломать/ставить блоки. 
Модуль world_cutted.py содержит несколько классов, которые позволяют удобно сохранять игру.
Модуль inventory.py содержит класс Inventory, в котором хранятся разрушенные блоки. Инвентарь также сохранятеся.
Модуль hero.py содержит класс Hero. В основном отвечает за анимацию действий персонажа.
Модуль buttons.py содержит класс кнопок, которые используются в меню игры. 
Модуль visual.py содержит класс Drawer и несколько вспомогательных функций, а также класс DrawableObject. Отвечает за прорисовку игры: персонажа, инвентаря и мира.
Вернёмся к геймплею. При запуске главного модуля main.py пользователь попадает в главное меню. При нажатии на кнопку "Start" генерируется новый мир; при нажатии на "Load save" можно выбрать сохранение. Чтобы выйти из меню сохранения, нужно нажать на Esc. Будьте внимательны: сохранений может быть всего три, поэтому, если попытаться добавить четвёртое, оно будет записано вместо самого старого из уже имеющихся. Чтобы выйти из игры нужно нажать на кнопку "Exit". 
В самой игре игрок может ходить, ломать и ставить блоки. Сломать можно почти любой блок, но попасть в инвентарь могут только три типа блоков: камень, земля и дерево. Эти блоки, соответственно, можно ставить. Чтобы выбрать блок нужно выбрать номер ячейки, в которой лежит блок. Из инвентаря можно выйти, нажав Esc.
При нажатии на кнопку Esc ставится пауза. Кнопка "Back" возвращает пользователя в игру. При нажатии на кнопку "Save game" пользователь может сохранить игру, введя название сохранение(пока что только на английском). При нажатии на Enter игра сохранится и пользователь автоматически попадёт в меню паузы. Чтобы вернуться без сохранения, нужно нажать Esc. Кнопка "To menu" возвращает пользователя в главное меню без сохранения текущего мира. 
Удачной игры!
